

🎯 Real-Time Quiz App
A simple, interactive quiz application built with HTML, CSS, and JavaScript. Test your knowledge with timed questions, track your score in real-time, and try to beat your high score!

''''''''

✨ Features

⏱️ Timed Questions – Each question has a countdown timer (15 seconds)

🎨 Instant Feedback – Correct answers turn green, wrong answers turn red

⏭️ Skip Questions – Move to the next question anytime using the Next button

🚫 Auto-Skip on Timeout – If time runs out, it automatically moves to the next question

📊 Live Score Tracking – See your score update in real-time

📈 Progress Bar – Visual indicator of how far you are in the quiz

🏆 High Score System – Your best score is saved using localStorage

📋 Detailed Results – See a breakdown of Correct, Wrong, and Not Attempted questions

🎨 Dynamic Themes – Background color changes for Start, Quiz, and Result screens

📱 Responsive Design – Works smoothly on mobile and desktop
